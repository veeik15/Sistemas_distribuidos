# Reflexion sobre la actividad
1. ¿Qué ventajas encuentras en trabajar directamente en la nube con GitHub?
   Que los cambios se guardan casi que de inmediato aunque toca darle a guardar previamente. Lo bueno es que tambien ves los cambios en tiempo real lo que es bueno. Ademas de poder trabajar en conjunto.
3. ¿Qué desafíos tuviste al crear el sitio sin usar herramientas locales?
   Se demoraba en mostrar los cambios, pero no mucho por lo que todo estuvo bien.
5. ¿Cómo podrías mejorar este sitio web en el futuro?
   Agregando correctamente una base de datos para guardar informacion, tambien mejorar el tipo de fuente y no queda menos agregar un incono, para que se vea mejor visualmente.
7. ¿Qué aprendiste sobre Sistemas Distribuidos al crear este sitio?
   Aprendí que los sistemas distribuidos mejoran la eficiencia y escalabilidad. Facilitando la alta disponibilidad y la tolerancia a fallos frente a sistemas monolíticos. Se gestionan mediante microservicios y almacenamiento    compartido, ocultando la complejidad subyacente al usuario final.
